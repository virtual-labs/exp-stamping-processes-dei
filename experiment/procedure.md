# Procedure

## Simulation Steps

### 1. Initialize
- Open the **Simulation** tab.
- The default view shows the cross-section of a **Die set**.

### 2. Select Material
- Choose a material from the dropdown menu (e.g., *Mild Steel*, *Aluminum*).
- Note the change in **Shear Strength**.

### 3. Set Parameters
- Set the **Thickness** of the sheet metal (0.5 mm to 5.0 mm).
- Adjust the **Die Clearance** as a percentage of thickness.
- Select the **Punch Speed**.

### 4. Run Simulation
- Click the **'Run Simulation'** button.

---

## Observation

- Watch the punch descend and the material deform.
- Observe the **Force vs. Displacement** graph plotting in real-time.
- Note the **'Max Force'** and **'Total Energy'** values displayed.

---

## Analysis

- Compare the force required for different materials and thicknesses.
- Observe how clearance affects the **"Fracture"** point in the animation.

---

## Export

- Download the data using the **'Download Data'** button for further analysis.
